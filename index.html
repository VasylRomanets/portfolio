About

I'm a Senior Unity Developer from Kyiv, Ukraine with 11+ years of professional experience building games across mobile, PC, and console. I care deeply about architecture, code quality, and creating experiences that feel polished to players.

At Moon Active since October 2019, I work on Pet Master and Travel Town — two of the most technically demanding Unity projects in mobile gaming, reached by millions of daily active users worldwide.

I designed the entire game architecture and touched everything: physics, game logic, UI, input systems, analytics, localization, and cloud saves.

I'm a strong believer in SOLID principles, clean architecture, and developer ergonomics. Outside of work I maintain open-source Unity tools used by developers worldwide, and I contributed fixes to DOTween — the most popular tween engine in the Unity ecosystem.

03
UnityLogWrapper
Open Source · ★ 6
Problem

UnityEngine.Debug calls execute in release builds, wasting CPU on low-end devices. Manually wrapping thousands of calls breaks Editor double-click navigation.

Solution

Drop-in wrapper in a separate assembly using [Conditional("LOGGING")] — logs compile out entirely in release. Editor still navigates to original call sites.

Impact

Zero-friction integration (one folder drop). Eliminates all log overhead in production builds with no changes to existing call sites.
